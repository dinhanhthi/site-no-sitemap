# site-no-sitemap

Simple static website (home + 4 child pages) without sitemap.

## Pages

- `/` (home)
- `/about.html`
- `/services.html`
- `/blog.html`
- `/contact.html`

## Deploy on Vercel

1. Push this repository to GitHub.
2. In Vercel, import the repo as a new project.
3. Keep default settings (no build command needed for static files).
4. Deploy.

## Notes

- There is no `sitemap.xml` in this project.
- Home page contains links to all 4 child pages.
